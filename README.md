# Merrickville Cookies Delivery

Single-file web app for order building, invoicing, delivery tracking, and driver
receipts. All prices CAD, 13% HST, delivery always free.

- Live app: https://crepracon.github.io/Merrickville-cookies-delivery-app/
- Live sync across devices via Firebase Realtime Database (configure in Settings, gear icon)
- Works offline / local-only until sync is configured
- Architecture notes and known landmines: see CLAUDE.md

No build step. `index.html` is the entire app.
